# Source Index — tomb-of-annihilation

**Source:** *Tomb of Annihilation* (Wizards of the Coast, 2017). **Imported:** 2026-06-23.

**Important — how source detail is handled in this campaign:**
The full module is a copyrighted commercial product and is **not** stored in this repo. The campaign's committed files (`world.md`, `world-nodes.md`, `arc.md`, `npcs*.md`, `state.md`) are **derived prep** — structure, summaries, and DM notes. For exact boxed text, room keys, maps, and stat blocks, the DM reads from the **player's own copy** of the module, one chapter at a time, as each scene comes into play (the lazy-corpus pattern, sourced from the upload rather than committed text).

Because the cloud session is ephemeral, the uploaded module file does **not** persist between chats. The derived campaign **does** persist (it's in the repo). For the richest play in a future session, the player can re-upload their module file; otherwise the derived prep above is sufficient to run the campaign.
